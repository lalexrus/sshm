# sshm
Консольный менеджер SSH с удобным разделением хостов по группам 

Консольный менеджер ssh соединений, основанный на содержимом ~/.ssh/config.
Поддерживает разделение хостов по группам  - название хоста (Host в config) просто указывайте по шаблону Группа-Хост.
Поддерживает использование ключей ssh (указанных в config)
Поддерживает keepassxc-cli и кеширование мастер-пароля базы keepass. Пароль ищется в базе по совпадению названия хоста (Host) и названия карточки в базе, в случае если карточка найдена, пароль копируется на 15 секунд в буфер обмена (поддерживается и wayland и X - достаточно измениь clip_cmd в начале скрипта)
есть #ai кодинг 
Для выбора хостов используется sk.
